# HOU Bookmark Project

  필요에 의해 찾게된 여러사이트를 즐겨찾기에 추가해도 시간이 지나면 어떤 즐겨찾기인지 파악하기 힘든 문제로 해당 프로젝트를 시작하게 되었습니다.
  필요한 페이지의 URL을 등록하면 페이지 내용을 분석하여 중요한 Keyword를 자동 추출하게 됩니다.
  자동 추출된 키워드로 어떤 사이트인지 확인하기 쉽고 찾기도 쉽게 만들어 보았습니다.
  또한 Trello의 카드 뷰 형식을 모방하여 보다 편하게 한눈으로 확인할 수 있습니다.

  현재 서비스는 Beta 서비스 입니다.
   http://52.36.205.200
